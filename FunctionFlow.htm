Function Flow Project

Function Flow aims to explore compilers' opportunites to task parallelism. We believe current programming languages and compilers should and can be more proactive in improving productivness of parallel programming. Function Flow achieve this by

  • providing a new language extension which unify expressing parallelsim,
  • static analyzer to report concurrency bugs,
  • automatic optimizer for task parallelism,
  • runtime which can leverage parallelism information.
